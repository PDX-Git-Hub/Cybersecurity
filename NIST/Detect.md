# Detect (DE)

**Purpose:**  
Identify cybersecurity events in a timely manner.

## Key Practices
- Enable logging of system and network activity
- Monitor authentication attempts and failures
- Use intrusion detection systems (IDS)
- Implement file integrity monitoring
- Set up alerts for suspicious behavior

## Example Controls
- Security Information and Event Management (SIEM) tools
- Regular log review procedures
- IDS/IPS systems monitoring network traffic
- Automated alerts for unusual activity

## Why It Matters
Faster detection minimizes damage and allows organizations to respond before attacks escalate.
